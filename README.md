# My DBMS

Project description:This is the very first project I have written. 
The idea is that you can create a user (by specifying name and age), 
search by name, clean up the database and get a list of all users.

## Installation

1. Download and install Python 3.x.
2. Download the source code for the project.
3. Open a terminal and navigate to the project directory.
4. Install the dependencies by running the command "pip install -r requirements.txt".

## Contributing to the project

If you would like to contribute to the project, you can do the following

1. Rework the Python code
2. Improve the interface of the website.
3. Change the database sqlite->SQLAlchemy
